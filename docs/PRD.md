# Drum Notes — Product Requirements (Index)

Drum Notes is a drum-notation editor focused on simplicity and speed: a drummer
can listen to a song and quickly transcribe its drum patterns without complex
music-notation software. The product prioritizes simplicity, speed and an
exclusive focus on drums.

This document is an **index**. The detail lives in the structured product docs
below — there is intentionally no duplicated content here.
